# Lessons Learned

A journal of mistakes and their prevention rules. Claude reads this at the start
of every session via the @lessons.md import in CLAUDE.md.

## Format

Each entry has:
- Date (YYYY-MM-DD)
- Short title
- Mistake (what went wrong)
- Correction (what the user said)
- Prevention rule (the concrete rule to prevent recurrence)
- Checklist item (one line to add to pre-flight checks)
